# Multisim-Discrete-Buck
基于 Multisim 的分离元器件 DC-DC 降压电路仿真,以及PCB设计方案
1. 项目亮点分立设计：未使用专用集成开关电源芯片，完全由 TL431 基准、LM358 运放及分立 MOSFET 搭建。闭环控制：引入电压负反馈，通过滞回比较器提升 PWM 生成的抗干扰能力
2. 电路分为四大核心模块：基准电压源：由 TL431ACD 产生的 2.5V精准参考电压 6666。+1误差放大器：使用 LM358AD 比较反馈电压与基准电压，进行线性放大 7。PWM 生成模块：由弛张振荡器产生高频锯齿波，通过滞回比较器生成稳定矩形波脉冲 888。+1Buck 功率模块：IRFZ44N 开关管结合 1N5822 续流二极管及 18mH 储能电感 9999。5. 调试记录与 FAQ
Q1: 为什么使用滞回比较器？
A: 在调试中发现，普通单线比较器在非线性区输出波形呈三角状而非矩形 。引入滞回特性虽略微增加了静态误差，但显著提高了波形稳定性 。
Q2: 仿真初期波形为什么先下降？
A: 这是由于电路通电瞬间的暂态响应及反馈环路的“建立过程”。电感电流不能突变，电容需要时间充电，反馈系统在 1-2ms 内完成闭环锁定 。
